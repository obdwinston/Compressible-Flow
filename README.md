**Domain Parameters**

| Parameter     | Value                             |
| :---:         | :---:                             |
| $L_x$         | $\tilde{x} \in$ [-1.0, 1.0]       |
| $L_y$         | $\tilde{y} \in$ [-1.0, 1.0]       |
| $L_t$         | $\tilde{t} \in$ [0.0, 10.0]       |
